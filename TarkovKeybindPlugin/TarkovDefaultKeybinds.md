# Escape From Tarkov — Default Keybinds (Consolidated)

## Movement & Stance Controls
- Move right: `D`
- Move left: `A`
- Move forward: `W`
- Move backward: `S`
- Smoothly lean right: `D` + `LAlt`
- Smoothly lean left: `A` + `LAlt`
- Lean right: `E`
- Lean left: `Q`
- Walk: `Caps`
- Sprint: `LShift`
- Jump: `Space`
- Vaulting: `Space`
- Crouch: `C`
- Prone: `X`
- Sidestep right: `E` + `LAlt`
- Sidestep left: `Q` + `LAlt`
- Overhead blind fire: `W` + `LAlt`

## Weapon & Interaction Controls
- Right side blind fire: `S` + `LAlt`
- Next walk pose: `Mouse Scroll+` + `C`
- Previous walk pose: `Mouse Scroll−` + `C`
- Shoulder transition: `Mouse4`
- Next: `Mouse Scroll+`
- Previous: `Mouse Scroll−`
- Mount weapon: `V`
- Fire: `LMB`
- Aim: `RMB`
- Reload weapon: `R`
- Quick reload: `R`
- Switch fire mode: `B`
- Set fire mode to full auto: `B`
- Check fire mode: `B` + `LAlt`
- Next magazine: `Mouse Scroll+` + `R`
- Previous magazine: `Mouse Scroll−` + `R`
- Detach magazine: `R` + `LAlt`

## Weapon Handling & Inspection
- Check ammo: `T` + `LAlt`
- Check chamber/Fix malfunction: `T` + `LShift`
- Unload chamber: `R` + `LCtrl`
- Switch between sights: `RMB` + `LCtrl`
- Change scope magnification: `RMB` + `LAlt`
- Scope elevation up: `PgUp`
- Scope elevation down: `PgDn`
- Scope zoom in: `Mouse Scroll+` + `LAlt`
- Scope zoom out: `Mouse Scroll–` + `LAlt`
- Inspect current weapon: `L`
- Fold or unfold stock: `L` + `LAlt`
- Toggle bipod: `V` + `LCtrl`
- Weapon on sling: `2`
- Weapon on back: `3`
- Melee weapon: `U`
- Sidearm: `1`
- Sidearm quick swap: `1`
